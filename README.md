# BlogHive

A modern blog platform built with **React** and **Appwrite**, focusing on clean architecture and scalable frontend patterns.

---

## Tech Stack

- **React**
- **React Router**
- **Redux Toolkit**
- **React Hook Form**
- **Tailwind CSS**
- **TinyMCE**
- **html-react-parser**
- **Appwrite (BaaS)**

---

## Features

- User authentication
- Create, update, and delete blog posts
- Rich text editor for content creation
- Image upload support
- Protected routes
- Modular service-based architecture

---

## Setup

```bash
npm install
npm run dev
```

## Deprecated vs New Syntax in Appwrite

- Collections -> Tables
- Attributes   -> Columns
- Documents -> Rows
